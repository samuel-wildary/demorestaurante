# Bella Italia - Atendimento WhatsApp Inteligente 🍕

Este projeto é uma demonstração interativa de um atendente virtual de restaurante via WhatsApp, utilizando a API da OpenAI com **Function Calling (Tools)** em tempo real.

O assistente (Bella) atua de forma humanizada e gerencia um carrinho de compras a partir da fala natural do cliente.

## 🚀 Tecnologias Utilizadas
- **React** (Vite)
- **Tailwind CSS**
- **Lucide Icons**
- **Canvas Confetti** (Efeitos visuais)
- **OpenAI API** (Recursive tool calling)
- **Web Audio API** (Sons sintéticos de notificação do WhatsApp)

## 📦 Como Rodar Localmente
1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
3. Abra `http://localhost:5173/` no seu navegador.
